## Steps to run:    

1. Copy the `.war` file from `WAR file` folder.  
2. Paste it in `\xampp\tomcat\webapps`  
3. Launch the xampp and start `Apache`, `Tomcat`.  
4. Open any browser and go to `https://localhost/Mail`
